Module 1 - Computer Systems (2021 MOD002580 TRI1 F01CAM)

This module introduces students to the components present in modern computer systems and networks. On completing this module, students will be able to specify, construct and maintain networked computer systems, and gain an in-depth understanding of common computer and network architectures, their function and confidently solve their problems.

Theoretical topics include CPU architectures; bus systems; types of memory and data; program execution; number systems; peripheral and network architectures, models and protocols; the importance of standards; network devices and cabling.

The module will be delivered in weekly lectures, which should be consolidated through significant self-study.

Laboratory sessions will enable students to gain the practical skills needed to construct, maintain and solve problems on networked computer systems.

Underpinning the theory students will learn (among other practical skills) how to safely assemble PCs (taking account of health and safety issues); construct (and test) network cables; examine low-level internet traffic and set up and configure local area networks. Student progress is assessed via in-class tests and via submitted coursework which may include a written assignment.

Module 2 - Introduction to System Architecture and Automation (2021 MOD007355 TRI1 F01CAM)

Welcome to "Introduction to System Architecture & Automation". This module is designed to help you develop a good understanding of the structure of common computer systems and how to automate networking and security aspects of these systems. You will learn about the "LAMP Stack" which is probably the most commonly used system architecture deployed on commercial networks. You will also be introduced to the Python programming language which is used to automate networking and security tasks.

There are many practical examples and tutorials in this module so you can gain experience in System Architecture & Automation that you will use in future modules and when you work in your chosen field.
